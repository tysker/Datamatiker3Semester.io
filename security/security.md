### JWT - Json Web Token

#  Basics:

* It is a Json object
* self contained. Carries all the information neccesary within itself.
* Mainly used in Web application

___


**Can be passed as...**

* a part of an URL (query string)
* form body parameter
* cookie
* HTTP Header (x-access-token)
