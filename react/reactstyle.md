## React Style


1. In tags: 

        <div style= { {backgroundColor: '' } } >
        
        
2. As a const variable:

        const itemStyle = backgroundColor: '';
        <div style= { itemStyle }>

3. In a seperate css file.

        .{
          background-color: '';
        }
